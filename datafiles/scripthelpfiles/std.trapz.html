trapz

Integration of a 1D function or a set of discreet data using trapezoidal algorithm.

Syntax:

trapz(vec_x=, vec_y=) → number

trapz(f=, a=, b=) → number

trapz{f= , a= , b= , inter=100 } → number

 

Arguments:

f A unary function, f(x)
a,b: Limits of the integral
inter: Number of intervals in the limits [a,b]. Default value is 100
vec_x, vec_y: Vectors

 

Examples:

>>x=std.linspace(0, 3)
>>y=x^2

>>std.trapz(x, y)
9.00046

 

Using a function:

>>std.trapz(function(x) return x^2 end, 0, 3)
9.00045
