logfit

Fits the given dataset to equation of y=a*ln(x)+b

Syntax:

logfit(x=, y=) → Vector
logfit{x=, y=} → Vector

where x and y are Vectors.
The function returns a vector containing the coefficients of the fitting.

 

Examples:

>>x=std.util.tovector{1,2,3,4}
>>y=x^2
>>std.logfit{x=x, y=y}
10.1506     -0.564824     COL

corresponding to: y=10.1506*ln(x)-0.564824
