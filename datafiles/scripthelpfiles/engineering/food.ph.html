Food:pH

Returns the pH value. By default it is 6.0.

Syntax:

ph(arg) → number

Notes:

  1. If arg is not provided, then returns the pH value.
  2. If arg is a number, then ph=num regardless of composition.
  3. If arg is a function, then ph=func(food)

Example:

>>f=Food.new{water=50}
>>f:ph()
6

>>f:ph(3)
>>f:ph()
3
