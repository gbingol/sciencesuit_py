ppois

Computes the cumulative (left-tailed) probability.
Returns number/Vector/Array.

Syntax:

1) ppois{q=, lambda=}

2) ppois(q=, lambda=)

 

q: Quantile ← number/Vector/Array
lambda: The average number of events per interval (event rate) ← number

Example Usage

>>std.ppois{q=2, lambda=3}
0.42319
