besselj

Computes the cylindrical Bessel function of the first kind.

Syntax:

besselj(arg, order)

where arg can be a number or an iteratable container
and order is the order of the Bessel function

Example:

>>t={0, 0.5, 1, 1.5, 2}

>>std.besselj(t, 0)
1    0.93847    0.765198    0.511828    0.223891

>>std.besselj(t, 1)
0    0.242268    0.440051    0.557937    0.576725
