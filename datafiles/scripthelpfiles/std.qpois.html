qpois

Computes the lower tail quantile (inverse of ppois).
The function always rounds up the result.

Returns number/Vector/Array.

Syntax:

1) qpois{p=, lambda=}

2) qpois(p=, lambda=)

 

p: Cumulative probability ← number/Vector/Array
lambda: The average number of events per interval (event rate) ← number

Example:

>>std.ppois{q=1, lambda=3}
0.199148

>>std.qpois{p=0.20, lambda=3} -- slightly greater than 0.199
2
