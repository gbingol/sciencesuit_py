Matrix:greater_equal

On each element, first performs an equality test and if it fails then checks whether the element is greater.

Syntax

greater_equal(arg, tol=1E-5)

arg: A number or a matrix
tol: tolerance level. Default is 1E-5.

 

Example:

>>m=std.util.tomatrix{ {1, 2, 3} , {1.1, 2.1, 3.1} , {7, 8, 9} }
>>m
1    2    3
1.1    2.1    3.1
7    8    9

>>m:greater_equal(2)
0    1    1
0    1    1
1    1    1
