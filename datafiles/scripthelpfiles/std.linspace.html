linspace

Generates n evenly spaced points in a given interval, [a,b].

Syntax:

  1. linspace(a=, b=, n=)
  2. linspace{a=, b=, n=}

 

a, b: Lower and upper limit of the interval, respectively. ← number
n: Number of data points in the interval ← integer

Example:

>>std.linspace{a=-2, b=2, n=6}
-2    -1.2    -0.4    0.4    1.2    2    COL

 

Note:See also, std.sequence
