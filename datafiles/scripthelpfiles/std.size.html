size

Finds the size of the given container.

Syntax:

size(container) → integer(s)

Containers:

Matrix, Range, Worksheet: Number of rows and columns
Vector: Length
Workbook: Number of Worksheets

 

Example:

>>std.size(std.activeworkbook())
2

>>range=Range.new(std.activeworkbook(), "Sheet 1!A1:B5")
>>std.size(range)
5    2

>>m=std.rand(3, 4)
>>std.size(m)
3    4

>>v=std.rand(4)
>>std.size(v)
4
