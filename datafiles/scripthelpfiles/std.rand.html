rand

Generates a uniformly distributed random numbers. Uses std.runif as its backend.
If both m and n are provided returns a Matrix, otherwise returns a Vector.

Syntax:

rand(m,[n=])

where both m and n must be of type integer.

 

Example:

>>m=std.rand(2,2)
>>m
0.68600 0.73003
0.54965 0.59908

>>v=std.rand(3)
>>v
0.515041    0.802075    0.272456    COL
