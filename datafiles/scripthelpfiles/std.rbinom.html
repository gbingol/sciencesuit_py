rbinom

Generates binomiallly distributed random numbers.
Returns a Vector or a number.

Syntax:

1) rbinom(n=, size=, prob=)

2) rbinom{n=, size=, prob=}

 

n: Number of random numbers ← integer
size: Number of independent trials ← integer
prob: Probability of success on each trial [0,1] ← number

Example Usage:

>>v=std.rbinom{n=10, size=5, prob=0.3}
>>v
0   2    2    1    0    2    0    2    2    4    COL
