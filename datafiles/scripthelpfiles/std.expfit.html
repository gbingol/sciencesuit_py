expfit

Fits the given dataset to equation of y=a*eb*x

Syntax:

expfit(x=, y=, [intercept=]) → Vector
expfit{x=, y=, [intercept=]} → Vector

where x and y are Vectors.
The function returns a vector containing the coefficients of the fitting.

 

Examples:

>>x=std.util.tovector{1,2,3,4}
>>y=2.1*std.exp(3.4*x)

>>std.expfit{x=x, y=y}
2.1     3.4     COL

corresponding to: y=2.1*e3.4*x
