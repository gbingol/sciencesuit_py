lm

Performs multiple linear regression.
Returns a Vector containing coefficients and a Lua table with extra info such as p-value.

Syntax:

lm (yobs=, factor=, InterceptIsZero=false)

yobs: Response ← Vector
factor: Factors ← Matrix/Vector
InterceptIsZero: Intercept ← boolean

 

Example:

>>coeff , RegresTable=std.linest(yobs, factors)

>>coeff
85.9008    -5.33    31.1028    COL

>>RegresTable
pvalue=1.16877e-08
Fvalue=21.4374

MSregression=530909

regression=1.06182e+06
