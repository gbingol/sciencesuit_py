Matrix:less

Tests whether the elements of the Matrix is less than a number (mi,j<a) or
elements of the Matrix is less than the elements of another Matrix (m1i,j<m2i,j).

Syntax

less(arg)

arg: A number or a matrix

 

Example:

>>m=std.util.tomatrix{ {1, 2, 3} , {1.1, 2.1, 3.1} , {7, 8, 9} }
>>m
1    2    3
1.1    2.1    3.1
7    8    9

>>m:less(2)
1    0    0
0    0    0
0    0    0
