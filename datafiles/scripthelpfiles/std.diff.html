diff

Computes the first order difference or the differential with respect to the first argument.
Returns Vector/Array.

Syntax:

diff(x=, y=nil)

Notes:

  1. If both parameters are Arrays, returns an Array. Otherwise, returns a Vector
  2. If y=nil, the differences (xi+1-xi) of the given vector are computed.
  3. If two arguments are provided, then the derivative (dy/dx) is computed.
    At the end points the function uses forward and backward differences
    and elsewhere the derivative is computed using central differences.

Example:

>>v=std.util.tovector{1, 1, 3, 2}
>>v
1    1    3    2    COL

>>std.diff(v)
0    2    -1    COL
