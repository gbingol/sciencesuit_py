mode

Finds the most frequently occuring element/elements in an iteratable container.
If mode does not exist returns nil, for unimodal returns a number and for multimodal a Lua table.

Syntax:

mode(Container=)

Examples:

>>t={1, 2, 3, 4, 5}
>>std.mode(t)
nil

>>t={1, 3, 5, 7, 9, 3, 5, 7, 3, 6}
>>std.mode(t)
3

>>v=std.util.tovector{1, 3, 5, 5, 7, 9, 3, 5, 7, 3, 6}
>>std.mode(v)
3   5
