solve

Solves a set of linear equations of Ax=b.

Syntax:

solve (A, b) → Vector

where argument A is a matrix containing the coefficients and b is the right-hand side vector.

 

Example:

Solve the following set of linear equations.
x + 2y =5
2x + 3y =11

--A={{1,2}, {2,3}} and b={5,11}

>>std.solve(A,b)
7    -1    COL

which means x=7 and y=-1.
