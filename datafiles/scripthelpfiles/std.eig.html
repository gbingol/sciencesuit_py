eig

Computes the eigenvalues and eigenvectors.

Syntax:

eig(m=, EigenVectors=false)

Argument m must be a square matrix. Very high performance computation can be performed if m is a symmetric matrix.

 

Notes:

  • If m is symmetric:
    • The function returns a Vector of eigenvalues.
    • if EigenVectors=true, the function returns a Vector and a Matrix: eigenvalues and eigenvectors.

    •  

  • If m is NOT symmetric:
    • The function returns a matrix of eigenvalues.The eigenvalues matrix is n by 2,
      where 1st column is the real and 2nd column is the imaginary part.
    • if EigenVectors=true, the function returns 2 matrices: eigenvalues and eigenvectors.
      eigenvalues: is n by 2, where 1st column is the real and 2nd column is the imaginary part.
      eigenvectors: is n by 2n, where m(n,1) and m(n,2) is the real and the imaginary part of the eigenvector for the first eigenvalue and so on .

Example:

>>m=std.util.tomatrix{ {1,3}, {2,4}}
>>m
1    3
2    4

>>std.eig(m)
-0.372281     0
5.37228     0

The eigenvalues are λ1=-0.372281+0i and λ2=5.37228+0i
