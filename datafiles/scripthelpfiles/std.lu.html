lu

Finds LU decomposition of a given matrix.

Syntax:

lu(A, method="lu")

Arguments:

A: Square or rectangular matrix
method: "lup" returns l, u and permutation matrix.
"lupq" returns l, u and two permutation matrices.

 

Example:

>>m=std.util.tomatrix{{10, -7, 0}, {-3, 2, 6} , {5, -1, 5} }
>>m
10 -7 0
-3 2 6
5 -1 5

>>l,u=std.lu(m)
>>l
1 0 0
-0.3 -0.04 1
0.5 1 0

>>u
10 -7  0
0 2.5 5
0 0 6.2
