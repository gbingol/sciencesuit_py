rpois

Generates random sample for the Poisson distribution.
Returns a Vector or a number.

Syntax:

1) rpois{n=, lambda=}

2) rpois(n=, lambda=)

 

n: Number of random numbers ← integer
lambda: Average number of events per interval (event rate) ← number

Example Usage:

>>v=std.rpois{n=10, lambda=3}
>>v
5    4    3    4    2    4    5    2    2    0    COL
