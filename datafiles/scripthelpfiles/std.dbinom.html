dbinom

Computes the probability value for Binomial distribution.
Returns Array / Vector / number.

Syntax:

1) dbinom{x=, size=, prob=}

2) dbinom(x=, size=, prob=)

 

x: Number of successes ← Array / Vector / number
size: Number of independent trials ← integer
prob: Probability of success on each trial [0,1] ← number

Example Usage:

>>std.dbinom{x=2, size=5, prob=0.3}
0.3087
