cumtrapz

Computes the left-tailed cumulative area and returns a column Vector.
Returns Vector/Array.

Syntax:

1) cumtrapz(f=, v1=) → Vector/Array

2) cumtrapz(v1=,v2=) → Vector/Array

3) cumtrapz{f=, a=, b=, inter=10} → Vector

f: A unary function ← function
a,b: Limits of the integral ← number
inter: Number of intervals in the limits [a,b]. Default value is 10 ← integer
v1, v2: Vector/Array, values on x- and y-axis, respectively.

 

Example:

>>std.cumtrapz{f=function(x) return x^2 end, a=1, b=5, inter=4}
0    2.5    9    21.5    42    COL
