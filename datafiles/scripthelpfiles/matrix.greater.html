Matrix:greater

Tests whether the elements of the Matrix is greater than a number (mi,j>a) or
elements of the Matrix is greater than the elements of another Matrix (m1i,j>m2i,j).

Syntax

greater(arg)

arg: A number or a matrix

 

Example:

>>m=std.util.tomatrix{ {1, 2, 3} , {1.1, 2.1, 3.1} , {7, 8, 9} }
>>m
1    2    3
1.1    2.1    3.1
7    8    9

>>m:greater(2)
0    0    1
0    1    1
1    1    1
