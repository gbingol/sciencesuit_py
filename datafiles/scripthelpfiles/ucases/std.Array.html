Array

Array is a 1D data structure and has the following properties:

  1. Can contain real (float, integer) and complex numbers and also non-numeric (string, nil) values.
  2. The nil value is usually meant to show the absence of a value.
  3. When an array is part of aDataFrame, then its status is called as linked.
  4. Mathematical operations (+, -, *, /, ^) are defined for arrays.

 

Creating Array

Array.new

Array.new(n=, [value=]) → Array

where n is the size of the array and value is the initial value to be assigned.

>>arr=std.Array.new(3)
>>arr
nil   nil   nil

>>arr=std.Array.new(3, "a")
>>arr
"a"  "a"  "a"

>>arr=std.Array.new(2, 3.14)
>>arr
3.14   3.14
